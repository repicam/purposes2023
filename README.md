# PURPOSES 2023 FULLSTACK WEB APPLICATION

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Dependencies

- Axios
- Bcrypt
- Cookie
- Jose
- Jsonwebtoken
- Moment
- Mongoose
- Semantic-ui-css|react
